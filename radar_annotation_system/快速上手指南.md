## 第1步：环境准备 🛠️

### 1.1 检查电脑系统
- Windows 10 或更高版本 ✅
- 至少8GB内存 ✅
- 至少5GB硬盘空间 ✅

### 1.2 安装Python（如果没有的话）
1. 访问 https://www.python.org/downloads/
2. 下载Python 3.8或更高版本
3. 安装时⚠️ **一定要勾选"Add Python to PATH"** ⚠️

### 1.3 验证Python安装
打开命令提示符（Win+R，输入cmd），输入：
```bash
python --version
```
如果显示版本号，说明安装成功！

## 第2步：安装软件 📦

### 2.1 下载项目文件
1. 确保所有文件都在 `radar_annotation_system` 文件夹中
2. 文件夹应该包含以下内容：
   ```
   radar_annotation_system/
   ├── main.py              # 主程序
   ├── run.py               # 启动脚本
   ├── requirements.txt     # 依赖列表
   ├── src/                 # 源代码文件夹
   ├── configs/             # 配置文件夹
   └── docs/                # 文档文件夹
   ```

### 2.2 安装必要的包
打开命令提示符，进入项目文件夹：
```bash
cd D:\desktop\test\radar_annotation_system
```

然后输入以下命令安装依赖：
```bash
pip install -r requirements.txt
```

⚠️ **注意**：这个过程可能需要10-20分钟，请耐心等待！

### 2.3 验证安装
安装完成后，输入：
```bash
python run.py --mode check
```
如果看到 "所有依赖项检查通过!" 说明安装成功。

## 第3步：启动系统 🚀

### 3.1 启动应用
在命令提示符中输入：
```bash
python run.py --mode app
```

### 3.2 等待启动
系统会显示：
```
雷达数据可视化与智能标注系统
========================================
启动雷达标注系统...
浏览器将自动打开应用界面
```

### 3.3 打开浏览器
- 浏览器会自动打开 http://localhost:8501
- 如果没有自动打开，手动在浏览器中输入这个地址

### 3.4 成功界面
你应该能看到一个标题为 "📡 雷达数据可视化与智能标注系统" 的网页！

## 第5步：实际案例 🎯

让我们用一个完整的例子来学习如何使用系统！

### 案例：标注汽车目标

#### 步骤1：数据处理

1. **选择数据源**
   - 在左侧边栏选择 "示例数据"

2. **生成雷达数据**
   - 在"雷达数据"部分点击 "生成示例雷达数据" 按钮
   - 系统会显示："成功处理 XX 个雷达点"

3. **生成图像数据**
   - 在"图像数据"部分点击 "生成示例图像" 按钮
   - 会看到一张示例图像

4. **进入下一步**
   - 点击顶部的 "点云可视化" 标签页

#### 步骤2：查看点云

1. **选择可视化方法**
   - 选择 "Matplotlib 2D"（最简单）
   - 调整点大小：设为 3
   - 颜色方案：选择 "强度"

2. **生成可视化**
   - 点击 "生成融合可视化" 按钮
   - 会看到一个2D点云图

3. **进入下一步**
   - 点击 "多模态融合" 标签页

#### 步骤3：多模态融合

1. **查看相机参数**
   - 展开 "相机参数设置" 查看（使用默认值）

2. **融合设置**
   - 勾选 "显示点云投影"
   - 勾选 "显示边界框"

3. **生成融合**
   - 点击 "生成融合可视化" 按钮
   - 会看到图像+点云的融合效果

4. **进入下一步**
   - 点击 "智能标注" 标签页

#### 步骤4：智能标注

1. **配置检测器**
   - 勾选 "启用 YOLO 检测器"
   - 其他检测器保持关闭状态

2. **开始检测**
   - 点击 "开始检测" 按钮
   - 等待处理完成

3. **查看检测结果**
   - 系统会显示检测到的目标数量
   - 每个目标包含：类别、置信度、边界框位置

4. **开始融合标注**
   - 点击 "开始融合标注" 按钮
   - 系统会生成3D标注结果

5. **查看融合结果**
   - 展开每个标注查看详细信息：
     - 3D中心点坐标
     - 3D尺寸
     - 置信度
     - 数据来源

#### 步骤5：导出结果

1. **选择导出格式**
   - 选择 "JSON" 格式（最常用）

2. **选择导出内容**
   - 勾选所有可用的项目

3. **导出结果**
   - 点击 "导出结果" 按钮
   - 点击 "下载JSON文件" 保存到电脑

### 🎉 恭喜！你已经完成了一个完整的标注流程！

## 🔧 常见问题解决

### Q1: 启动时提示"端口被占用"
**解决方案：**
```bash
python run.py --mode app
```
等待几秒后重试，或重启电脑。

### Q2: 安装依赖时出现红色错误
**解决方案：**
```bash
pip install --upgrade pip
pip install -r requirements.txt --no-cache-dir
```

### Q3: 可视化窗口不显示
**解决方案：**
- 检查是否安装了显卡驱动
- 尝试使用不同的可视化方法
- 确保数据已正确加载

### Q4: 检测结果为空
**解决方案：**
- 降低置信度阈值（设为0.3）
- 检查图像是否清晰
- 尝试使用示例数据测试
