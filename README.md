# NUAA-162510415-Radar-Data-Visualization-Exploration-and-Intelligent-Annotation
## 雷达数据可视化探索与智能化标注
*16院-2025-天目启航-雷达数据可视化探索与智能化标注*

### 课题简介
随着自动驾驶、智能安防等技术的快速发展，毫米波雷达因其抗干扰能力强、全天候工作的特性，逐渐成为关键传感器之一。然而，由于数据样本有限、人工标注成本高等问题，如何高效地对毫米波雷达数据进行可视化与智能化标注，成为亟需解决的技术难题。本课题旨在通过研究雷达数据的可视化技术和智能化标注方法，提升毫米波雷达数据的可读性、可用性与标注效率。研究内容主要包括以下两个方面：
1.	雷达数据的可视化
o	对毫米波雷达信号进行数据预处理，包括噪声过滤与点云数据的生成；
o	实现点云数据的三维可视化展示；
o	探索毫米波雷达与可见光图像的多模态融合展示，提升毫米波雷达数据的直观性与可理解性。
2.	雷达数据的智能化标注
o	利用现有的大语言模型或多模态模型，对可见光图像中的关键目标（如建筑物、信号塔等）进行定位与识别；
o	将雷达点云与图像数据结合，实现三维关键物体快速定位；
o	研究自动化物体标注技术，并结合人工校验机制，实现高精度的半自动标注流程。
________________________________________
### 技术指标
1.	实现毫米波雷达点云的三维可视化展示，并完成雷达数据与图像数据的融合展示，提供相应的案例与代码；
2.	开发一个关键物体智能化标注系统，支持对关键障碍物（如汽车、建筑物、路标等）的半自动标注，并结合人工校验机制完成高精度标注。
